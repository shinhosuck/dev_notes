YPES OF HTTP REQUEST METHODS:
1. GET
2. HEAD
3. POST
4. PUT
5. DELETE
6. CONNECT
7. OPTIONS 
8. TRACE 
9. PATCH

HTTP STATUS RESPONSE CODES:
1. (2xx) Success:
    -200 - Success
    -201 - Created
    -202 - Accepted

2. (3xx) Redirections:
3. (4xx) Client Error
    -404 Not Found
4. (5xx) Server Error 
    -500 Internal Server Error
